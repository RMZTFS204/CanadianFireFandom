About This Project

A firefighter fandom site built as part of Emerging Multimedia Practice 2.

Project Purpose

This website is a firefighter fandom project that highlights Canadian fire trucks, firefighting gear, history, and basic fire safety tips.

It was created to combine my personal interest in firefighting with my multimedia skills in web design, layout, animation, and structured content.

Skills Demonstrated

  • Responsive web design using HTML, CSS, and JavaScript
  • Consistent visual design with a clear colour palette and typography
  • Simple animations for navigation and page transitions
  • Organized content about real Canadian firefighting vehicles
